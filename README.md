# SDDL_Converter
A quick Python script to convert SDDL strings into English. 

## Installtion

Run: 

`wget https://github.com/DanielIsaev/SDDL_Converter`

Or simply download the script manually. 


## Usage 

`./sddl_convert.py <sddl_string>`
